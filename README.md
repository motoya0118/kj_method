# README
## 開発言語
ruby 3.0.4
Rails 6.1.7

## アプリケーションに取り入れる技術(就業Termで学んだもの)
1. AWS EC2
2. devise
3. Ajax

## アプリケーションに取り入れる技術(カリキュラム外)
1. Vue.js
2. vuedraggable

## アプリケーションの実行手順
```:terminal
git clone git@github.com:motoya0118/kj_method.git
cd kj_method
bundle install
rails db:create db:migrate
rails webpacker:install
rails s
```
